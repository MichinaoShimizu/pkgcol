# pkgcol

## install

```bash
python3 -m pip install -r requirements.txt
```

## execusion

```bash
mbp:pkgcol michinao$ python3 pkgcol -h
usage: pkgcol [-h] -r [REPOSITORY_NAMES ...] [-f [FILTER_WORDS ...]] [-c] [-s]

optional arguments:
  -h, --help            show this help message and exit
  -r [REPOSITORY_NAMES ...], --repository_names [REPOSITORY_NAMES ...]
  -f [FILTER_WORDS ...], --filter_words [FILTER_WORDS ...]
  -c, --clean_after_deps
  -s, --skip_repository_setting
```
